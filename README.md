# HIITtimer
独学iOSアプリ1作目

## このアプリの目的
トレーニングのHIIT専用で、
<br>
30秒の間に20秒トレーニング、10秒休憩というルールのタイマーを、
<br>
音声で教えてくれるようにした。

## このアプリの概要
トレーニングのHIIT専用のタイマー機能アプリ。
<br>
タイマーとトレーニングメニューの設定が可能。


## 開発環境
* Xcode: 13.2.1
* iOS: 15.0
* Deployment Target: 14.0
